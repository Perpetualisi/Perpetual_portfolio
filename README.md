## 💡 Setup Instructions

1. Clone the repository:
    ```bash
    git clone https://github.com/Perpetualisi/Perpetual_portfolio.git
    ```

2. Navigate into the project folder:
    ```bash
    cd Perpetual_portfolio
    ```

3. Install dependencies:
    ```bash
    npm install
    ```

4. Run the project locally:
    ```bash
    npm run dev
    ```
